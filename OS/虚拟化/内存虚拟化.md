Every address generated by a user program is a virtual address!

Yes, the Operation System 封装了所有的细节

隔离是非常重要的，

两个实体相互隔离，意味着其中一个的失败不会影响到另外一个。

一些操作系统实现让某些部分和操作系统的其他部分相互隔离，所谓「微内核」，这样比「整体内核」拥有更大的可靠性。

所谓透明，就是对于程序来说不可见，
实现透明有很大的好处，但是也需要付出代价。

虚拟内存VM virtual memory 的目标

1. 透明
2. 效率
3. 保护
