## 输入 

一个冷知识，
如果理解next() && nextLine() 的区别呢？
next排除任何空前缀，也不吸收空后缀，不能携带带有空格的字符串
nextLine只认换行符，
这会带来什么问题呢？

如果你的字符串输入在一行当中，使用空格分割，
我建议直接使用next逐一捕获
或者，不建议了。

但是，要记住，next捕获之后，要使用nextLine吞掉换行符，
这一般不会构成大问题，
除非你没有注意到！！！
