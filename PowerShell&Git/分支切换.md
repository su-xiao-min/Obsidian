我希望，效果就是，把我的指针回到dev，接着checkout一个分支，保留我上上次commit之后的所有更改。

简单地说，

直接



```bash
git log --online -n 3 
git checkout -b feature/new  origin/dev
git cherry-pick bc32234

git push --set-upstream origin feature/new

git checkout feature/old 
git reset --hard HEAD~1
git push --set-upstream origin feature/old


```

但是需要注意一个小问题。

就是，你没有提交的东西，

在你`git reset --hard` 的时候会丢失，所以，

要么就把它一起commit到某个分支上面，

要么，就干脆把它

`git stash` 起来，

所以，

应该小心一点。
