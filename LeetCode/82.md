```txt
给定一个已排序的链表的头 head ， 删除原始链表中所有重复数字的节点，只留下不同的数字 。返回 已排序的链表 。
//
// 
//
// 示例 1： 
// 
// 
//输入：head = [1,2,3,3,4,4,5]
//输出：[1,2,5]
// 
//
// 示例 2： 
// 
// 
//输入：head = [1,1,1,2,3]
//输出：[2,3]
// 
//
// 
//
// 提示： 
//
// 
// 链表中节点数目在范围 [0, 300] 内 
// -100 <= Node.val <= 100 
// 题目数据保证链表已经按升序 排列 
// 
//
// Related Topics 链表 双指针 👍 1433 👎 0
```

不是很难的题目

## 1

```
        ListNode dummy = new ListNode(0, head);
        for (ListNode cur = dummy;
             cur != null && cur.next != null;
             cur = cur.next) {
            ListNode next = cur.next;
            int val = next.val;
            ListNode nNext = next.next;
            if (nNext == null) return dummy.next;
            int nVal = nNext.val;
            if (val != nVal) continue;
//                进入最后一个逻辑，这时候所有相同的节点都需要移除，找到第一个不相同的节点，
//                置为cur.next,当然因为它也可以为null，所以这里需要在前面添加判断。
//                这里会在中途修改cur，所以不应该使用for，
            while (next.next != null && val == next.next.val) {
                next.next = next.next.next;
            }
            cur.next = next.next;
        }
        return dummy.next;
```



丑陋的解法，并且错误。

## 3

```java
ListNode dummy = new ListNode(0, head);
ListNode cur = dummy;
while (cur.next != null && cur.next.next != null) {
    int val = cur.next.val;
    if (cur.next.next.val == val) {
        while (cur.next != null && cur.next.val == val) {
            cur.next = cur.next.next;
        }
    } else {
        cur = cur.next;
    }
}
return dummy.next;
```

优雅永不过时，
非常简单，使用最少的变量，哪怕出来cur.next.next也好，总比我的next.next.next要好。

## 总结

对比一下，
为什么本题需要dummy，因为本题的head也可能不会保留。

为什么本题需要next.next，因为仅仅保证cur指向的东西是稳定的，这就足够了。
