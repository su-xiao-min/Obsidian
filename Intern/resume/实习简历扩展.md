**项目描述**：

TeamERP 是一个企业资源管理系统，实现从人员申报、考核、入职、任务下派到考评的全流程，同时包括企业内容的电子资源管理、财务管理。

**责任描述**：作为助理研发人员，主要负责开发文件管理系统、消息推送支持。

**项目亮点**：
- 采用 JWT + Redis 双令牌机制，通过 Access Token 处理业务请求，通过 Refresh Token 实现用户无感的令牌刷新。在保证安全性的同时优化用户体验。
- 采用MinIO进行文件管理，通过MD5作为文件唯一标识，在上传文件时会自动校验MD5值，对于已经存在的文件则采用逻辑上传，提高文件传递效率。
- 基于WebSocket实现员工任务实时推送，监测所有会改变task的事件，发现任务改变时自动向Web端与手机端推送消息。
