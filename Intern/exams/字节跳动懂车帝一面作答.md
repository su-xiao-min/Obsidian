# 4️⃣ BIO / NIO / AIO

你答得太抽象。

---

## 正确区分：

| 模型 | 阻塞       | 说明             |
| ---- | ---------- | ---------------- |
| BIO  | 阻塞       | 一个连接一个线程 |
| NIO  | 同步非阻塞 | 多路复用         |
| AIO  | 异步非阻塞 | 回调             |

NIO不是“轮询避免休眠”。

> I/O多路复用：一个线程监听多个FD
