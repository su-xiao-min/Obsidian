简单地说

<kbd>Ctrl</kbd>+<kbd>Alt</kbd>+其他的键，就可以映射到不同的重构上面

例如

- <kbd>P</kbd>，对应的是参数重构，换言之，把方法内的一个值转移到参数当中，这样
- <kbd>N</kbd>，内联重构
- <kbd>V</kbd>，变量提取
- <kbd>L</kbd>，格式化，超级好用的东西，它能不能实现eslint的效果呢？

另外，还可以使用右键查看本地的历史记录，简单地说，就是非常轻量的版本控制，

应该不需要通过git来实现，

git的特色在于能力强大，

但是操作起来有一点麻烦。

<kbd>Alt</kbd>+<kbd>Shift</kbd>+<kbd>C</kbd>，就可以快快乐乐查看本地的修改了。

<kbd>Ctrl + P</kbd>，可以实现查看形参。

<kbd>Ctrl + Q</kbd>也是大大的好方法啊，可以查看方法符号的文档。

超级好用的几个快捷键

<kbd>Shift</kbd> + <kbd>Shift</kbd>，任意搜索

<kbd>Ctrl</kbd>+ <kbd>N</kbd>，搜索类

<kbd>Ctrl</kbd>+<kbd>Q</kbd>，展开文档

<kbd>Esc</kbd> ，关闭弹出的窗口

<kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Shift</kbd>+<kbd>N</kbd>，寻找方法和全局变量



<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>F</kbd>，在项目文件当中查找字符串的存在，

这也是我认为非常有价值的地方。
